PAUSECtrl ¶

   <task pointer> = PAUSECtrl( "<pause-callback>", "<resume-callback>")

   function <pause-callback>()
     -- 引数なし
   end

   function <resume-callback>()
     -- 引数なし
   end

引数 ¶

  • <pause-callback> プロセスがバックグラウンド化される直前に呼ばれるコールバックの名前。不要な場合は nil を指定する。
  • <resume-callback> プロセスがフォアグラウンド化される直前に呼ばれるコールバックの名前。不要な場合は nil を指定する。

戻り値 ¶

  • <task pointer> 生成された PAUSECtrl タスクのポインタ

解説 ¶

ゲームアプリケーションが端末の機能によってForeground/Backgroundを切り替えられる際に送られるOSイベントを検出し、 コールバックを呼び出す。

コマンド ¶

(コマンド無し)
